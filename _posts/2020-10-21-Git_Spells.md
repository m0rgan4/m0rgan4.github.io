Para ver las ramas:

`git branch`

Para crear una rama:

`git checkout -b nombredelarama`

Para seleccionar una rama:

`git checkout nombredelarama`

Para añadir todos los archivos modificados para el commit:

`git add .`

Para hacer un commit:

`git commit -m "motivo del commit"`

Para subir al repositorio:

`git push`
